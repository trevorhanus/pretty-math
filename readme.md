# Pretty Math

## Contributing

### Calculator

**Run Locally**

```
// from the root dir
$ yarn install
$ yarn start:calculator
```

### Docs

**Run Locally**

```
$ cd website
$ npm run develop
```

### Tree Explorer

The Tree Explorer is a simple React app that is useful when working on the `math` module. It allows us to enter plain text math expressions are visually see the AST that the parser builds.

**Run Locally**

```
// from the root dir
$ yarn install
$ yarn start:explorer
```

### Tests

**Run tests**

```
# run tests
$ yarn test

# watch files during development
$ yarn test:watch

```
